HTML (HyperText Markup Language) is used for front-end development of a website. HTML defines the meaning and the structure of the web content.

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. With this, developers can create simple or complex visual design that brings colourfulness to a website.

JS (JavaScript) is a lightweighy, interpreted compiled programming language with first-class functions. it is a scripting language used to instroduce logic to a website.
